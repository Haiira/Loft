开发工具

Unity

Unity官方网站

下载与设置

前往官方网站下载Unity Hub，这是一个Unity版本管理工具，用于管理工程和各版本的Unity引擎。这是3.6.1版本下载安装后的界面：

点击左上角的齿轮（Preference）> Appearance > Language 可以修改中文界面，之后进入安装量（Installs）界面即可选择需要的Unity版本进行安装。

需要注意的是，Unity项目对引擎版本的一致性要求比较严格，即同一个项目，用不同版本的引擎打开可能会出现问题，因此建议不要频繁更换（或直接不要更换）一个项目的Unity版本。

选择所需版本安装后，Unity Hub会跳出添加模块选项，可以在此处选择你需要随Unity一并安装的工具。其中一个默认勾选的选项是Microsoft Visual Studio Community 2019，这是C#代码开发工具，相关内容会在本文的第二部分详细讲解，为避免重复安装，这里请取消勾选。其他选项维持默认。
另外，模块中有Unity引擎内部的中文支持（虽然汉化还不完全），如果需要可以在该界面选择安装。个人建议是不要汉化。

模块可以在Unity安装后另行添加，但Hub仅提供安装服务，卸载需要在文件资源管理器内手动删除，比较麻烦，因此仅安装目前需要的模块即可。

Unity引擎的安装位置和新项目的默认位置都可以在Hub的偏好设置（Preference）内进行修改，建议放置在便于寻找且磁盘空间足够的位置，请确保安装路径内没有中文，以免日后开发中出现问题。
之后就是漫长的下载安装了。

Visual Studio

Visual Studio官方网站

相当专业的代码开发工具，因为与Unity可以直接默认联动（比较方便），推荐使用。想要追求体积上的轻便也可以下载VS Code代替，但相对的也会失去一些Visual Studio提供的辅助功能。
可用的代码编写工具非常多，总体来说选择自己喜欢的软件即可，这里不多做赘述。
